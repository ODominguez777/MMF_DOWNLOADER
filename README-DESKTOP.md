# Bulk Downloader (Desktop v1.5.0)

Desktop app (Electron) + Bash/PowerShell scripts to bulk-download **your** MyMiniFactory library.

## Quick start (recommended)

1. `npm install` then `npm start`
2. **Open MyMiniFactory** → sign in → **Capture session**
3. **Auto load my IDs** (catalog loads in pages of 25 from the store API)
4. Remove any models you do not want from the list
5. **Execute Pipeline** (Step 1 metadata → Step 2 test → Step 2 full)

See also: [DESKTOP_EXE.md](DESKTOP_EXE.md) (build installer), [CHANGELOG.md](CHANGELOG.md).

## Rate limits (conservative)

| Step | Delay |
|------|--------|
| Metadata (Step 1) | 6 s between models |
| STL/ZIP (Step 2) | 5 s between files |
| Images | 3 s |
| Catalog pages | 1 s between store API pages |

## Resume after session expiry

- Step 1 skips existing `model_<id>.json` files.
- Step 2 skips files already on disk.
- **Windows packaging:** the installer ships `tools/zip.exe` (Info-ZIP 3.0) so clients do not need to install `zip` separately. Each `.rar`/`.7z` from MMF is wrapped in its own `.zip` (`file.rar` → `file.zip`). MMF-delivered `.zip` files are left as-is.
- If the session expires mid-run, capture a new session and re-run; completed work is kept.

## Auth

- Primary: embedded browser + **Capture session** (encrypted storage when OS keychain is available).
- Fallback: **Advanced → manual credentials** in the Configuration panel.
